Installation¶

Core functionality¶

TOPP-RA can be installed from source by the following steps:

git clone https://github.com/hungpham2511/toppra && cd toppra/
pip install -r requirements.txt --user
python setup.py install --user

These lines implement the most basic functionalities, such as parametrize trajectories subject to velocity and acceleration constraints. Some advanced functionalities require openrave, cvxpy and so on. See Installation for the full instruction.

(Optional) qpoases¶

It is possible to use qpoases instead of the default LP solver seidel as solverwrapper. Install qpOASES by entering the following commands in the terminal:

git clone https://github.com/hungpham2511/qpOASES
cd qpOASES/ && mkdir bin && make
cd interfaces/python/
pip install cython
python setup.py install --user

Advanced functionality¶

In order to run some of the examples, it is necesssary to install openRAVE. A good instruction for installing this library on Ubuntu 16.04 can be found here.

note: the humanoid examples are not maintained.

Building docs¶

The lastest documentation is available at https://hungpham2511.github.io/toppra/.

To build and view the documentation, install sphinx then run the following commands in the terminal

cd <toppra-dir>/docs/
make clean && make html
<browser> build/index.html

Testing¶

TOPP-RA uses pytest for testing, which can be installed from pip. To run all the tests, do:

cd <toppra-dir>/tests/
pytest -v
